# Unidade V - Conceitos avançados de POO

Esta unidade avança em recursos de orientação a objetos em C++. Os exemplos praticam herança múltipla, membros `protected`, classes amigas, métodos estáticos, atributos estáticos, funções virtuais, vínculo estático, vínculo dinâmico, classes abstratas e sobrecarga em exemplos pontuais.
